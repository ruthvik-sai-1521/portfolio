1. UPSC Preparation Assistant Web App

Overview:

This project is a dedicated web application designed to support students preparing for the UPSC examination. It serves as a centralized hub for accessing essential study resources, syllabus information, and previous year question papersâ€”all in a simple and user-friendly interface.

Key Features

1. Home Page:

Showcases a welcoming interface with my contact details. Provides easy access to previous year question papers. Users can select a year, view the paper directly in PDF format, and download it with a single click.

2. Material Page:

Contains categorized resources: NCERT textbooks (6th to 10th grade), Additional UPSC-related study materials. Clicking on the download buttons redirects users to Google Drive for seamless access and downloading.

3. Syllabus & Pattern Page:

Displays the complete UPSC syllabus. Explains the examination stages: Prelims, Mains, and Interview. Describes the types of questions asked in each stage and how the exam is structured.
